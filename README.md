# SED_game

An introductory WeChat mini game project for testing purposes

## 源码目录介绍

```
├── audio                                      // 音频资源
├── images                                     // 图片资源
├── .eslintrc.js                               // 代码规范
├── main.js                                    // 游戏入口主函数
├── game.js                                    // 游戏逻辑主入口
├── game.json                                  // 游戏运行时配置
├── project.config.json                        // 项目配置
└── project.private.config.json                // 项目个人配置
```
